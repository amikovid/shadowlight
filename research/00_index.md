# research/ — index

Reference docs that back every decision in the Volley landing page and the broader Chief Achiever submission. Each doc here exists so that, in the Loom and in the deck rationale sections, every word, color, and structural choice can be defended with a citable reason.

These docs are sources, not deliverables. Operational/working files (CLAUDE.md, CLAUDE_DESIGN_BRIEF.md, JUSTIFICATION_DOC.md, index.html) live at the repo root and reference docs in this folder by `research/<filename>`.

## How to use these in the Loom

The submission's hardest test is the Loom. Per Max: *"Decks tell me what you did. Loom tells me how you think."*

When narrating each part of the deliverable, the structure is:

1. **What I decided** (the choice)
2. **Why I decided it** (the source, point at one of these docs)
3. **What I'd test next** (kill criteria, alternative hypothesis)

That's not a script. It's a structure: **decision → citable reason → test plan.** Every section can be narrated this way.
